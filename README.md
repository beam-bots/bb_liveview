# BB.LiveView

Interactive LiveView-based dashboard for Beam Bots-powered robots.

## Installation

Add `bb_liveview` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:bb_liveview, "~> 0.3.1"}
  ]
end
```
